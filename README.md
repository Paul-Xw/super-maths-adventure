# Super Maths Platform v7

รองรับ Year 7, Year 8, Year 9 ในแพลตฟอร์มเดียว พร้อม Micro Games 4 แบบที่เล่นได้จริง

ติดตั้ง: Upload ทั้งโฟลเดอร์ขึ้น GitHub แล้ว Deploy ผ่าน Cloudflare

Supabase optional: Run sql/platform_v7_schema.sql

Curriculum map แก้ได้ที่ data/curriculum.json และควรตรวจเทียบกับ framework ที่โรงเรียนมีสิทธิใช้งาน
